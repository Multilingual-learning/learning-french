
Cinquième leçon

« Je suis grand, tu es petit. 
— Oui, je suis petit. » 

Monsieur, vous êtes petit ; Madame, vous êtes petite ; Mademoiselle, vous êtes petite.
Il est petit, elle est petite.

Nous somme petits. — Oui, vous êtes petits. 

Ils sont petits, elles sont petites.

Exercices

I) Conjuguez "être gros", au présent de l'indicatif.

  Je suis gros
  Tu es gros (grosse)
  Il est gros
  Elle est grosse
  Nous sommes gros
  Nous sommes grosse
  Vous êtes gros
  Ils sont gros
  Elles sont grosse
  
II) Complétez : Je suis grand. Tu es petit. Il est gros. Elle est mince. Nous sommes grands. Vous êtes petits. Ils sont gros. Elles sont minces. Les livres sont épais. Le crayon est long. Tu es grand. Vous êtes grand. Vous êtes grands.

III) Comment est...

  Comment est le crayon ? Crayon est court. 
  Comment est le mur ? Le mur est haut.
  Comment est le tableau ? Le table est large.
  Comment est la lampe ? La lampe est ronde.
  Comment est le plafond ? Le plafond est bas.
  Comment est le plancher ? Le plancher est rectangulaire.
  Comment sont les livres ? Les livres sont épais.
  Comment est la fenêtre ? La fenêtre est claire.
  Comment sont les portes ? Ils sont étroites.
  Comment est la porte ? Elle est étroite.
  Comment sont le crayons ? Ils ne sont pas courts. Ils sont longs.
  
IV) Écrivez le contraire : 

  Le mur est haut, le mur est bas.
  La règle est longue, la règle est courte.
  Le cahier est épais, le cahier est mince.
  Le banc est étroit, le banc est large.
  
V) Écrivez le féminin : 
  
  Le livre épais, la gomme épaisse.
  Un long crayon, une longue règle.
  Le mur étroit, la table étroite.
  Le mouchoir blanc, la cravate blanche.
  Le cahier rouge, la serviette rouge.
  Le gros homme, la grosse femme.
  
IV) Est-ce que homme est un nom de personne ? 
    Ouais! Homme est un nom de personne.
  
  Est-ce que table est un nom de personne ? 
  Non, table n'est pas un nom de personne, il est un nom de la chose.
