
Troisième leçon

  Voici un crayon
  Voici un livre
  Maintenant le crayon est sur le livre.
  Voici une gomme
  Voici une serviette
  Maintenant la gomme est sur la serviette
  Les crayons sont sur le livre
  Les gommes sont sur la serviette
  
  --
  
  Où est le cahier ?      Il est dans la serviette.
  Où est la règle ?       Elle est sous le livre.
  Où est le professeur ?  Il est devant le tableau.
  Où est le tableau ?     Il est derrière le professeur.
  
EXERCICES

I) Écrivez le, la ou l' : 

  le livre
  la montre
  l'étudiant
  le sac
  le plafond
  la règle
  l'homme
  le professeur
  l'oiseau
  le tableau
  l'horloge
  le mur

II) Est-ce que ...

  Est-ce que le livre est sur la table ? Oui, il est sur la table.

  Est-ce que le cahier est sur la serviette ? Non, il est sous la serviette.
  
  Est-ce que la gomme est sous le cahier ? Non, il est sur la cahier.
  
  Est-ce que le professeur est derrière le tableau ? Non, il est devant le tableau.
  
  Est-ce que les étudiants sont dans la classe ? Oui, ils sont dans la classe.
  
  Est-ce que les étudiantes sont dans la classe ? Non, elles ne sont pas dans la classe.
  
III) Où est ... 

  Où est le cahier ? Le cahier est sous le crayon.
  Où est le crayon ? Le crayon est sur le cahier.
  Où est le professeur ? Professeur est derrière la chaise.
  Où est la chaise ? La chaise est devant le professeur.
  Où sont les livres ? Les livre sont sur le banc.
  Où est le banc ? Le banc est sous les livres.
  Où sont les clés ? Les clés sont dans le sac.
  
IV) Complétez : 

  Le stylo est sur le banc. 
  Le professeur est devant les étudiants. 
  Le tableau est derrière le professeur. 
  Les mouchoirs sont dans les sacs.
