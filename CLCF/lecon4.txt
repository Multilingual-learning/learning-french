
Quatrième leçon

le livre vert
la robe verte

  
  Singulier:
    Masculin                  Féminin
  
    Le livre est vert         La robe est verte
    Le livre est gris         La robe est grise
    Le livre est brun         La robe est brune
    Le livre est noir         La robe est noire
    Le livre est bleu         LA robe est bleue
    
  Pluriel:
    Masculin                  Féminin
    
    Les livres sont verts     Les robes sont vertes
    Les livres sont gris      Les robes sont gris
    
  Attention !
  
    Le livre est rouge   La robe est rouge
    Le livre est rose    La robe est rose
    Le livre est jaune   La robe est jaune
    Le livre est blanc   La robe est blanche
    
  De quelle couleur est la craie ? La craie est blanche.
  
  Est-ce que la craie est verte ? Non, la craie n'est pas verte; elle est blanche.
  
Grammaire 

  Un livre bleu    "un" est un article indéfini
                   "livre" est un nom
                   "bleu" est un adjectif qualificatif
                   
Exercices

I) De quelle couleur...

  De quelle couleur est le tableau ?    Le tableau est noir.
  De quelle couleur sont les bancs ?    Les banc sont bruns.
  De quelle couleur sont les murs ?     Les murs sont blancs.
  De quelle couleur est la porte ?      La porte est grise.
  De quelle couleur sont les cahiers ?  Les cahiers sont gris.
  De quelle couleur sont les robes ?    Les robes sont de couleurs différents.
  Est-ce que la craie est blanche ?     Oui, la craie est blanche.
  Est-ce que le tableau est blanc ?     Non, le tableau n'est pas blanc, il est noir.

II) Écrivez les adjectifs féminins : 
 
  Le banc est brun, la table est brune. 
  La cahier est vert, la gomme est verte.
  Le stylo est noir, la serviette est noire.
  Le gant est gris, la clé est grise.
  Le mouchoir est blanc, la craie est blanche.
  Le crayon est rouge, la robe est rouge.
  Le livre est bleu, la règle est bleue.

III) Écrivez les adjectifs pluriels : 

  Les crayons sont gris. 
  Ce sont des livres rouges.
  Les robes sont noires.
  Les gants sont bruns.
  Les murs sont blancs.
  Ce sont des sacs noirs.
  Ce sont des cravates vertes.
  Ce sont des mouchoirs bleus.

IV) Écrivez les adjectifs masculins : 
  
  La robe est verte, le gant est vert.
  La cravate est jaune, le sac est jeune.
  La clé est grise, le mur est gris.
  La gomme est blanche, le mouchoir est blanc.
  La serviette est rouge, le livre est rouge.
  La règle est noire, le stylo est noir.
  
V) Écrivez des adjectifs de couleur :

  La gomme est verte.
  La table est blanche.
  Le calendrier est gris.
  La règle est brune.
  Les cahiers sont jaunes.
  Les cravates sont bleues.
  La craie est noire.
  Les livres sont blancs ou rouges ou gris ou bruns.
  Les robes sont blanches ou rouges ou gris ou brunes.
  
VI) Où est ...

  Où est le cahier ?      Il est dans la serviette.
  Où est la gomme ?       Elle est sur la table.
  Où est le sac ?         Le sac est sous le banc.
  Où est le tableau ?     Il est derrière le professeur.
  Où sont les livres ?    Ils sont dans la bibliothèque.
  Où sont les lampes ?    Les lampes sont sous la plafond.
  Où sont les chaises ?   Les chaises sont dans la classe.
  Où est l'horloge ?      L'horloge est dans la mur.
  Où sont les clés ?      Les clés sont dans la chambre.
